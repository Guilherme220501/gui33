Engenharia de Software
Faltam
Modelagem de processos
Refere-se à representação visual ou descritiva dos passos, atividades e fluxos de trabalho de um processo organizacional, com o objetivo de analisar, melhorar e otimizar sua eficiência.
Comportamento organizacional
Estuda como indivíduos e grupos se comportam dentro de uma organização, abordando fatores como motivação, liderança, cultura e dinâmica de equipe para promover um ambiente de trabalho produtivo
Engenharia de Produtos
Envolve o desenvolvimento e a gestão de produtos, desde a concepção até a produção, focando em atender às necessidades do mercado e garantir a viabilidade técnica e econômica.
Engenharia de Usabilidade
Concentra-se em projetar produtos e sistemas que sejam intuitivos e fáceis de usar, avaliando a interação do usuário para melhorar a experiência e a satisfação do cliente.
Gerencimento de projetos
Refere-se à aplicação de conhecimentos, habilidades, ferramentas e técnicas para planejar, executar e controlar projetos, assegurando que sejam concluídos dentro do prazo, orçamento e escopo estabelecidos.
Big Data em Python:
Envolve o uso da linguagem de programação Python para manipular, analisar e visualizar grandes volumes de dados, utilizando bibliotecas como Pandas, NumPy e frameworks como Apache Spark para processamento de dados em larga escala.
Banco de dados
É um sistema organizado para armazenar, gerenciar e recuperar dados de forma eficiente, utilizando modelos como relacional (SQL) ou não relacional (NoSQL) para atender a diferentes necessidades de armazenamento e consulta.

Linguagens


Linguagens
Python
Uma linguagem de alto nível, versátil e fácil de aprender, amplamente usada em ciência de dados, automação e desenvolvimento web.
Java
Uma linguagem orientada a objetos, popular por sua portabilidade e robustez, utilizada em aplicações empresariais e desenvolvimento Android.
Javascript
Linguagem de programação essencial para desenvolvimento web, permitindo criar interatividade e dinamismo em páginas da web.
C
Uma linguagem de programação de baixo nível, fundamental para o desenvolvimento de sistemas e software embarcado, conhecida pela sua eficiência e controle de hardware.
C++
Uma extensão do C que adiciona orientação a objetos, amplamente utilizada em desenvolvimento de jogos, aplicações gráficas e sistemas de alto desempenho.
C#
Linguagem de programação desenvolvida pela Microsoft, utilizada principalmente no desenvolvimento de aplicações para Windows e em plataformas como Unity para jogos.
Ruby
Conhecida pela sua simplicidade e produtividade, Ruby é uma linguagem orientada a objetos, popular para desenvolvimento web, especialmente com o framework Ruby on Rails.
PHP
Uma linguagem de script do lado do servidor, amplamente utilizada para desenvolvimento web dinâmico e criação de sistemas de gerenciamento de conteúdo como WordPress.
